# 🚀 Vera App (Flutter Streak Tracker)

**Developed by Afaq Ahmad**

A beautifully designed Flutter application that helps users stay consistent with daily exercises using a **streak tracking system**, engaging animations, and a modern UI.

---

## 🌟 Features

* 🔥 Daily streak tracking system
* 🎉 Animated streak celebration (Lottie + custom animations)
* 📅 Calendar-based activity visualization
* 💪 Daily workout suggestions
* 🎨 Multi-brand theming support (Vera, Endo-Plan, Symptome)
* ⚡ Smooth animations with `AnimationController`
* 📱 Clean and responsive Material UI (Material 3)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/afaqahmad/flutter-streak-app.git
```

2. Navigate into the project:

```bash
cd flutter-streak-app
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

---

## 🎯 How It Works

* User completes a daily exercise
* Streak count increases
* Celebration animation is triggered
* UI updates dynamically with progress

---

## 🎨 Branding System

The app supports multiple brands using a configurable `AppBrand` class:

```dart
AppBrand currentBrand = AppBrands.vera;
```

You can easily switch themes for different apps.
